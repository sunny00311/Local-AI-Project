# Model Files

## Required Model

Please copy your downloaded model file here:

**File:** `qwen2.5-0.5b-instruct-q4_0.gguf` or `qwen2.5-0.5b-instruct-q4_k_m.gguf`

**Size:** ~350MB

**Download from:** [HuggingFace - Qwen2.5-0.5B-Instruct-GGUF](https://huggingface.co/Qwen/Qwen2.5-0.5B-Instruct-GGUF/tree/main)

## Instructions

1. Download the Q4_0 or Q4_K_M quantized model
2. Copy the .gguf file to this directory
3. The file will be bundled with your app during build

## Current Status

Check if the model file exists:
```bash
dir assets\models\*.gguf
```

If you see the file listed (~350MB), you're ready to proceed!
